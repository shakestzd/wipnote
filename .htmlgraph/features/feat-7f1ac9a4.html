Description

Forward partner of 'htmlgraph blame'. Walks every tracked source file, aggregates by track via feature_files → features → contained_in chains, and emits a per-track inventory (with optional inverse --by-file mode). Modes: --format text|json|markdown. Surfaces an 'untracked' bucket for files with no feature attribution (gap signal). Replaces the manually-written docs/code-areas.md once shipped — CI step regenerates docs/code-areas.md and fails on diff. Depends on 'htmlgraph blame' existing because they share the underlying query helper; build blame first, then layer code-areas on top.
