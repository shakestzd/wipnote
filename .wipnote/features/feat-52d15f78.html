Embedded ttyd terminal in dashboard — MVP

Done Medium Priority

Implemented In:

  • session c89a0910-ba83-4833-9dee-5364fab28c1a

Description

MVP that lets users open an interactive Claude session from the dashboard, scoped to the current work item. Approach: Go server spawns ttyd on a localhost port running 'htmlgraph claude', returns the port, frontend iframes it. Minimal surface: POST /api/terminal/start (spawn ttyd, return port+pid), POST /api/terminal/stop (kill pid), a dashboard 'Terminal' button that opens a modal iframe. If the call includes a work item ID, run 'htmlgraph feature start ' in the same shell before 'htmlgraph claude' to scope the session.
