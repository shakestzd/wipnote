Description

Add a self-contained writer service owned by `wipnote serve` first, with an internal abstraction that can later move to a separate `wipnote daemon` if needed. The service holds the migrated writable DB connection and serializes event/index SQLite mutations through a bounded queue. Producers submit best-effort derived-index updates only after canonical persistence has succeeded; SQLite is never the sole durable copy.
