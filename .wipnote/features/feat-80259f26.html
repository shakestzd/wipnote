Description

Move schema migrations out of the unconditional `Open` body and into an ordered migration runner guarded by `PRAGMA user_version`. On warm open when `user_version == currentSchemaVersion`, skip CREATE INDEX, ALTER TABLE, DROP TABLE, trigger creation, copy-and-swap, and `NormalizePlanFeedbackValues`. On cold/new DB creation, create the full current schema and set the version. On old DBs, run only missing migrations in order.
