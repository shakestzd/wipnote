Gate dashboard terminal feature behind HTMLGRAPH_TERMINAL env var (default off)

Done Medium Priority

Implemented In:

  • session cb5d73c7-d3c1-4598-9768-a07cf69db974

Description

The dashboard terminal feature (button opens ttyd sidecar sessions via /api/terminal/*) ships unconditionally today. Gate it behind HTMLGRAPH_TERMINAL=1: when unset, (a) serve.go skips registering the four /api/terminal/* routes and (b) the dashboard hides the open-terminal button + launcher modal entry point. Frontend should auto-detect the backend being disabled (probe /api/terminal/sessions on init; hide UI on 404) so the two halves stay in sync. Reversible by setting the env var.
