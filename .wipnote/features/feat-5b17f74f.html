wipnote search subcommand (ast-grep wrapper)

Todo Medium Priority

Description

Add 'wipnote search ' that wraps ast-grep with agent-friendly output. Validates ast-grep binary on PATH at command time; clear error if missing. Optional --lang flag for language hint. Output is one match per line: file:line: . Caps results at 50 by default with --limit override. Phase 2 (wipnote refs) deferred to follow-up.
