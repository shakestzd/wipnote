Description

htmlgraph hook pretooluse and posttooluse crash with SIGBUS inside modernc.org/sqlite/lib._walFindFrame on ARM64 (Codespaces devcontainer). Stack: libc.AtomicLoadNUint16 → _walFindFrame → _getPageMMap. Reliably reproducible after prolonged session / concurrent htmlgraph serve. Candidates: (1) switch DB to journal_mode=DELETE or TRUNCATE to bypass WAL mmap; (2) bump modernc.org/sqlite past v1.37.1 — ARM64 atomic-load codegen has churned; (3) swap to mattn/go-sqlite3 (CGo) to drop the libc.a_load_16 path entirely. Blocks /htmlgraph:plan (PostToolUse blocking errors on every write) and every htmlgraph CLI command that opens the DB. See plan-251676c5 draft.
