Description

htmlgraph serve and the pretooluse hook crash with SIGBUS inside modernc.org/sqlite _walFindFrame via _getPageMMap. Filesystem is fakeowner-over-overlay (Docker Desktop macOS virtiofs bind mount to /workspaces), which does not support SQLite's mmap-backed page reads. Fix: set Pragmas[mmap_size]=0 in internal/db/pragmas.go (default to disabled; mmap buys marginal gains over WAL+page-cache). Follow-up: autodetect unsafe fstypes (fuse*, virtiofs, 9p, overlay upper) in Open() and only enable mmap on native filesystems.
