Description

Session nodes render at 60% of nodeRadius (d.type === 'session' ? nodeRadius(d) * 0.6 : nodeRadius(d)) but the label wrapping and hub filter use the full nodeRadius. Labels think they have more room than the circle actually provides and spill onto the background. Fix: unified visualRadius(d) helper used by circle render, collision force, hub filter, and wrapTextInCircle calls — single source of truth.
