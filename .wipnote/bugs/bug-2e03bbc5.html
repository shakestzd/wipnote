Description

devcontainer.json declares Docker named volumes over .htmlgraph/ and .claude/, which are git-tracked directories containing canonical work items and plugin config. The named volumes replace the git-cloned content with empty volume state, so htmlgraph find/status report 0 items and the wrong directory tree. Named volumes are for non-git-tracked artifacts (node_modules, .venv); git-tracked trees should come from the clone. Removes the mounts entirely in v0.54.8.
