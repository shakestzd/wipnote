Port 8088 was intended as a devcontainer-specific dashboard port (per the project memory note), but is being applied as the default in production builds too. Production users should see 8080 (cobra default) unless they explicitly pass --port. Investigation: grep for '8088' across cmd/ and internal/ outside of test fixtures and devcontainer docs; identify the leak point.
